# CADAgent - Fusion 360 Add-in

AI-powered CAD modeling assistant. Describe what you want in plain English, and CADAgent builds it in Fusion 360.

## Download

- **Windows**: [Download CADAgent-Windows.zip](https://github.com/er-fo/CADAgent/releases/download/v1.0-win/CADAgent-Windows.zip)
- **macOS**: [Download CADAgent-macOS.zip](https://github.com/er-fo/CADAgent/releases/download/v1.0-mac/CADAgent-macOS.zip)

## Installation

### Windows
Copy the `CADAgent` folder to:
```
%AppData%\Autodesk\ApplicationPlugins\
```

### macOS (Web Install)
Copy the `CADAgent` folder to:
```
~/Library/Application Support/Autodesk/ApplicationPlugins/
```

### macOS (App Store)
Copy the `CADAgent` folder to:
```
~/Library/Containers/com.autodesk.mas.fusion360/Data/Library/Application Support/Autodesk/ApplicationPlugins/
```

Restart Fusion 360. The add-in loads automatically.

## Usage

1. Open Fusion 360
2. Go to **Tools** > **Add-Ins** > **CADAgent** > **Run**
3. Type your request (e.g., "Create a 5cm cube")
4. Click **Execute**
