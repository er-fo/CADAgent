# CADAgent Resources

This folder contains resources for the CADAgent Fusion 360 add-in.

## Icon Files

Command icons should be placed in this directory with the following specifications:

- **Format**: PNG with transparency
- **Size**: 32x32 pixels for normal, 64x64 for high-DPI displays
- **Naming convention**:
  - `command_name_16.png` - 16x16 icon
  - `command_name_32.png` - 32x32 icon (default)
  - `command_name_64.png` - 64x64 icon (high-DPI)

## Current Icons

No custom icons are currently defined. Fusion 360 will use default icons for commands.

To add custom icons:
1. Create PNG files with appropriate sizes
2. Name them according to the command they represent
3. Place them in this folder
4. Update command definitions to reference the icon path
