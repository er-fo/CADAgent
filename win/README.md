# CADAgent for Windows

## Installation

1. Download and extract `CADAgent.bundle`

2. Copy the entire `CADAgent.bundle` folder to:
   ```
   %AppData%\Autodesk\ApplicationPlugins\
   ```

   Full path example:
   ```
   C:\Users\YourName\AppData\Roaming\Autodesk\ApplicationPlugins\CADAgent.bundle\
   ```

3. Restart Fusion 360

Fusion will detect the add-in automatically on launch.

## Verify Installation

1. Open Fusion 360
2. Go to **Tools** > **Add-Ins**
3. CADAgent should appear in the list
